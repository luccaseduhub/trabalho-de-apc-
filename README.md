# Sistema de Condomínio

Sistema de cadastro (CRUD) desenvolvido em C para gerenciamento de apartamentos.

## Sobre o projeto
O programa implementa:
- Armazenamento em vetor de `struct` (limite de 10 registros).
- Validação de entrada de dados (proteção contra caracteres inválidos e IDs duplicados).
- Lógica de repetição e decisão concentrada na função `main()`.

## Execução
Para executar, basta compilar o arquivo `main.c` em qualquer compilador C padrão e rodar o executável gerado.
